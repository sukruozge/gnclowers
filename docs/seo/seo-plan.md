# Aselovers — SEO / AEO Plan (örgü oyuncak · crochet toys)

Hedef: **"örgü oyuncak", "amigurumi oyuncak", "el yapımı bebek oyuncağı"** (TR) ve
**"crochet toys", "amigurumi toys", "handmade baby toys"** (EN) için Google ilk sayfa.

## Dürüst beklenti
- **Uzun-kuyruk + Türkçe terimler** (ör. "isme özel amigurumi", "organik bebek oyuncağı",
  "amigurumi nasıl seçilir") ilk sayfaya **birkaç ay içinde** girmek gerçekçi.
- **Kısa/rekabetçi head-term** ("crochet toys" gibi) uluslararası devlerle yarışır;
  aylar-yıl + **backlink (domain otoritesi)** ister. Buraya sabırla oynuyoruz.

## Yapıldı (canlı) ✅
- Teknik: sitemap (368 URL, Search Console'a gönderildi), hreflang (tr/en),
  robots AI-crawler'lara açık, `llms.txt` (AEO), yapısal veri (Organization, Product +
  gerçek Etsy puanı, FAQPage, BreadcrumbList), `max-image-preview:large`.
- On-page: ana sayfa + ürünler sayfası title/H1/meta head-term'lere göre ("Örgü Oyuncaklar
  & Amigurumi" / "Crochet Toys & Amigurumi"). Metinler doğal dile çevrildi (AI-kokusu giderildi).
- İçerik: 15 blog + **haftalık otomatik yeni yazı** (aşağıda).
- Varyasyonlar Etsy'den senkron (zengin ürün sayfaları).

## Sürekli içerik motoru (otomatik, bulutta) ✅
- Önceden yazılmış kaliteli yazılar `published:false` + `publishAt` ile kuyrukta.
- **`blog-publish.yml`** günlük cron: tarihi gelen yazıyı canlıya alır, bugünün tarihiyle
  yayınlar → Google düzenli "taze içerik" görür. LLM API anahtarı gerekmez.
- Kuyruk bitince yeni parti eklenir (Claude'a "blog kuyruğunu doldur" demen yeterli),
  ya da ileride Anthropic API anahtarı eklenip tam otomatik üretime geçilir.

## Sıradaki adımlar (öncelik sırası)
1. **Backlink / otorite** (en yüksek kaldıraç): Etsy mağaza profili + Instagram bio + Pinterest
   → siteye link. Örgü/anne blogları, Türk el sanatları dizinleri, "handmade Türkiye" listelerine
   kayıt. Birkaç kaliteli link büyük fark yaratır.
2. **Pinterest** (görsel arama = örgü nişi için altın): her ürün/blog için pin. API onayı beklemede.
3. **Google Business** gerekmez (online) ama **Google Merchant / ücretsiz ürün listeleri** değerli.
4. **İç linkleme**: her blog yazısından ilgili ürün/kategoriye link (drip yazılarında var).
5. **Bing Webmaster** + sitemap (Copilot/AI cevapları için).
6. **Ürün açıklamaları**: Etsy'den gelenler bazen düz; zamanla en çok satanları elle zenginleştir.

## KPI (Search Console'dan izle)
- 2-4 hafta: sayfalar indekslenir ("Sayfalar" sekmesi).
- 4-8 hafta: uzun-kuyruk sorgularda gösterim + ilk tıklamalar ("Performans").
- 3-6 ay: hedef terimlerde sıralama yükselişi; içerik + backlink ile ilk sayfa.
